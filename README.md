# Daniel Chen — Portfolio

Personal portfolio website for Daniel Chen, a student software engineer. Built to showcase projects and provide a point of contact for potential employers.

Live at [daniel-chen.dev](https://daniel-chen.dev)

## Tech Stack

- **Framework** — Next.js 14 (App Router, TypeScript strict, SSG)
- **Styling** — Tailwind CSS with CSS custom properties (dark/light theme)
- **Content** — MDX files parsed with `next-mdx-remote` + `gray-matter`
- **Animations** — Framer Motion
- **Email** — Resend API
- **Deployment** — Vercel
